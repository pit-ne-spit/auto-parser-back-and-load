## Документация модулей

Подробная документация по каждому из трех основных модулей:

1. [Модуль единоразового скачивания](MODULE_INITIAL_LOADER.md)
   - Первоначальная загрузка всех объявлений (~250,000 записей)
   - Пагинация, обработка дубликатов, продолжение после прерывания

2. [Модуль ежедневного обновления](MODULE_DAILY_UPDATER.md)
   - Получение изменений за предыдущий день (~15,000 записей)
   - Обработка типов изменений (added/changed/removed)
   - Merge JSON данных

3. [Модуль нормализации](MODULE_NORMALIZER.md)
   - Преобразование сырых данных в структурированный формат
   - Извлечение полей, преобразование типов, перевод текстов
   - Батчевая обработка с транзакционностью
