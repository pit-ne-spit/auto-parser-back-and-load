# Настройка офлайн перевода через Argos Translate

## Описание

**Argos Translate** - это полностью офлайн библиотека для перевода, которая не требует подключения к интернету после установки языковых пакетов.

## Преимущества

- ✅ **Полностью офлайн** - работает без интернета
- ✅ **Бесплатно** - не требует API ключей
- ✅ **Быстро** - переводы выполняются локально
- ✅ **Приватно** - данные не отправляются в интернет

## Установка

### Шаг 1: Установка библиотеки

```bash
pip install argostranslate
```

**Примечание:** Если возникают ошибки с блокировкой файлов, закройте все процессы Python и попробуйте снова.

### Шаг 2: Установка языковых пакетов

После установки библиотеки нужно установить языковые модели для перевода с китайского на русский:

```bash
# Обновить список доступных языков
python -m argostranslate.package --update-languages

# Установить пакет для прямого перевода китайский -> русский (если доступен)
python -m argostranslate.package install zh ru

# Если прямой перевод недоступен, используйте промежуточный язык (английский)
python -m argostranslate.package install zh en
python -m argostranslate.package install en ru
```

### Шаг 3: Проверка установки

Проверьте установленные языковые пакеты:

```bash
python -m argostranslate.package list
```

Вы должны увидеть установленные языки, например:
```
zh -> ru
или
zh -> en
en -> ru
```

## Использование

### Автоматическая установка через скрипт

Используйте готовый скрипт для установки:

```bash
python scripts/setup_argos_translate.py
```

### Использование в коде

После установки Argos Translate будет использоваться автоматически по умолчанию:

```bash
# Запуск итеративного процесса с Argos Translate
python scripts/iterative_translation_enrichment.py --translation-provider argos
```

Или просто запустите без указания провайдера (Argos будет использован по умолчанию):

```bash
python scripts/iterative_translation_enrichment.py
```

## Размер языковых пакетов

Языковые пакеты могут быть довольно большими (несколько сотен МБ каждый):
- Китайский -> Русский: ~200-500 МБ
- Китайский -> Английский: ~200-500 МБ  
- Английский -> Русский: ~200-500 МБ

Убедитесь, что у вас достаточно места на диске.

## Ограничения

1. **Качество перевода** может быть ниже, чем у онлайн сервисов (Google, DeepL)
2. **Первый запуск** может быть медленным (загрузка моделей)
3. **Требуется место на диске** для языковых моделей

## Альтернативы

Если Argos Translate не подходит, можно использовать другие офлайн решения:

1. **transformers + модели** - использование моделей машинного обучения напрямую
2. **MarianMT** - модели от Facebook для перевода
3. **Opus-MT** - открытые модели перевода

Но Argos Translate - самый простой вариант для начала.

## Troubleshooting

**Проблема: "Языковые пакеты не установлены"**
```bash
python -m argostranslate.package --update-languages
python -m argostranslate.package install zh en
python -m argostranslate.package install en ru
```

**Проблема: "Модуль не найден"**
```bash
pip install argostranslate
```

**Проблема: Ошибка при установке из-за блокировки файлов**
- Закройте все процессы Python
- Перезапустите терминал с правами администратора
- Попробуйте установить снова
